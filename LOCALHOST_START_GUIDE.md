## 🛠️ Troubleshooting

### If page doesn't load:
1. Check if server is running (see terminal output)
2. Try refreshing browser (Ctrl+R)
3. Clear browser cache
4. Check console for errors (F12)

### If resume analyzer fails:
1. Ensure spaCy model installed:
   ```bash
   python -m spacy download en_core_web_sm
   ```
2. Check file size (< 16MB)
3. Verify file format (.pdf, .docx, or .txt)

### If styles don't appear:
1. Hard refresh: Ctrl+Shift+R
2. Check browser console (F12)
3. Verify CSS files loaded
