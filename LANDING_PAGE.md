## 🔧 Customization Options

### Change Background Colors
Edit `landing.css`:
```css
--bg-dark: #YOUR_COLOR;
--bg-darker: #YOUR_COLOR;
```

### Adjust Orb Colors
```css
.orb-1 { background: radial-gradient(circle, YOUR_COLOR, transparent); }
```

### Modify Particle Count
Edit `landing.js`:
```javascript
const particleCount = 50; // Change this number
```

### Change Animation Speed
```css
animation: float 20s ease-in-out infinite; /* Change 20s */
```
