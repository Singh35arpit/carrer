## 🔧 Customization Options

### Change Orb Colors
Edit `dashboard-dark.css`:
```css
.orb-1 { background: radial-gradient(circle, YOUR_COLOR, transparent); }
```

### Adjust Glass Opacity
```css
background: rgba(255, 255, 255, 0.05); /* Change 0.05 */
```

### Modify Animation Speed
```css
animation: float 20s ease-in-out infinite; /* Change 20s */
```

### Increase Particles
Edit `landing.js` (used by dashboard):
```javascript
const particleCount = 50; // Increase this number
```
