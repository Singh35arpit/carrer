## 🔍 Browser-Specific Fixes

### **Chrome/Edge:**
```
1. Go to: chrome://settings/clearBrowserData
2. Select "All time"
3. Check everything
4. Click "Clear data"
5. Restart browser
```

### **Firefox:**
```
1. Go to: about:preferences#privacy
2. Scroll to "Cookies and Site Data"
3. Click "Clear Data"
4. Check everything
5. Click "Clear"
```

### **Safari:**
```
1. Safari → Preferences → Privacy
2. Click "Manage Website Data"
3. Click "Remove All"
4. Confirm
```
